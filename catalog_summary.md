# Catalog and Habitat Concepts

## Resource Catalog Entry

A resource catalog entry is an official Kepler definition of a possible resource type in the world. It describes what the resource is.

## Local Inventory

Local inventory is the set of resources the habitat actually owns. Inventory is habitat state, so it changes as the habitat gathers, uses, stores, or transfers resources.

## Blueprint

A blueprint is the Kepler definition of something that can be produced or built. It describes inputs, outputs, build time, prerequisites, and capabilities. 

## Module

A module is an installed habitat component, such as a battery, life-support unit, or workshop. It has its own identity, connections, capabilities, runtime attributes, and current state.
